# readme

## zus-cli

zus-cli command line utility for react on **[zus-cli](https://github.com/zhaozeq/init_react_project)**

### Getting started

```bash
## usage
npm install zus-cli -g

## create your project
$ zus init project-name
$ cd project-name
$ npm run dev

## start server
$ zus server

## build project
$ zus build
```
